# 🧠 GenAI & LLM Architecture

## 🔎 Retrieval Augmented Generation (RAG) Pipeline

```text
        ┌────────────────────┐
        │     User Query     │
        └─────────┬──────────┘
                  ↓
        ┌────────────────────┐
        │   Embedding Model  │
        │ (OpenAI / HF)      │
        └─────────┬──────────┘
                  ↓
        ┌────────────────────┐
        │   Vector Database  │
        │  (Pinecone / FAISS)│
        └─────────┬──────────┘
                  ↓
        ┌────────────────────┐
        │   Retriever Layer  │
        │    (LangChain)     │
        └─────────┬──────────┘
                  ↓
        ┌────────────────────┐
        │        LLM         │
        │ (GPT / Open Source)│
        └─────────┬──────────┘
                  ↓
        ┌────────────────────┐
        │   Final Response   │
        └────────────────────┘
```
